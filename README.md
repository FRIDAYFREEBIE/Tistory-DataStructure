# Data Structure Study Collection

### 📅 Study Period: 2025-08-02 ~ 2025-10-02

---

## 📌 Project Overview
- **Purpose**: To organize personal study materials by posting them on Tistory and documenting related resources
- **Goal**: To systematically document learned data structure concepts on both the blog and GitHub
- **Study Method**: Selecting data structures, implementing them manually for learning purposes, and writing blog posts about the implementation process

---

## 🔑 Topics Covered

- **Tree**
  - Binary Tree
  - Tree Traversal
  - Binary Search Tree (BST)
  - AVL Tree
  - AVL Tree Rotation

- **Graph**
  - Graph Structure and Concepts
  - Depth-First Search (DFS)
  - Breadth-First Search (BFS)

- **Implementation**
  - Implemented core data structures directly using C++
  - Practiced dynamic memory management and pointer-based structures
  - Studied time complexity and structural characteristics of each data structure

- **Algorithm Concepts**
  - Recursive tree traversal
  - Graph traversal using stack and queue
  - Self-balancing tree rotations
  - Search optimization through hierarchical structures

---

## 🤔 What I Learned

- Improved understanding of how fundamental data structures operate internally.

- Learned how trees and graphs are connected to real-world algorithms and game systems.

- Gained experience implementing data structures manually instead of relying on built-in libraries.

- Improved understanding of recursion, memory management, traversal algorithms, and balancing techniques.

- Learned how structural design impacts algorithm efficiency and time complexity.
